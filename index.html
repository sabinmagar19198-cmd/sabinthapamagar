About

Engineer. Builder. Curious.

I am a B.E. graduate in Electronics, Communication and Information Engineering from Kantipur Engineering College, affiliated to Tribhuvan University, Dhapakhel, Lalitpur.

My four-year programme covered digital and embedded systems, RF & microwave engineering, wireless communication, satellite communication, signal processing, computer networks, industrial automation, AI, and telecommunications — taught through both theory and extensive lab work.

What I value most is building hardware that works under real constraints. My final year Major Project — a web-based Augmented Reality eyewear try-on system using MTCNN and VGG-16 — achieved 92% face shape classification accuracy. My Minor Project brought IoT home automation to life using an ESP32, RFID, PIR sensors, and Arduino IoT Cloud monitoring. Both showed me that the most interesting engineering problems sit at the boundary of hardware and software.

I am motivated by the security implications of the devices we build. Embedded systems deployed at scale — in homes, factories, healthcare — are an increasingly critical attack surface, and that intersection of hardware engineering and cybersecurity is where I want to grow.

Projects

What I Have Built

Engineering projects from my degree at KEC — each solving a real problem with hardware, software, or both.

01
Major Project · EX755 · March 2025

Virtual Eyewear Try-On System

A web-based AR platform that analyses face shape and overlays eyewear frames in real time. MTCNN detects facial regions; VGG-16 classifies face shape into 5 categories (oval, round, square, heart, oblong); MediaPipe localises 468 landmarks for pupil distance calculation and frame alignment; Three.js renders 3D glasses on a live webcam feed. Achieved 92% face shape classification accuracy and a <5% AR overlay width error margin.

MTCNNVGG-16MediaPipeThree.jsFlaskMySQLWebRTCPython
Completed · March 2025Supervised by Er. Sachin Shrestha · KEC
02
Minor Project · EX 654 · February 2024

IoT Home Automation System

An ESP32-based smart home system integrating multiple sensors for autonomous appliance control and cloud monitoring. PIR & LDR sensors manage automatic lighting; RFID authentication secures door locks; MQ2 gas sensor provides kitchen fire detection; water level sensors manage tank pump automation. All sensor data streams to Arduino IoT Cloud for real-time remote control. Simulated in Proteus before hardware deployment.

ESP32RFIDPIR SensorLDRMQ2Water Level SensorArduino IoT CloudProteus
Completed · February 2024KEC · Dept. of Computer & Electronics Eng.

Competition Project · HEX-2023

Autonomous Robo Race Vehicle

Designed and built an autonomous robot for the HEX-2023 Robo Race at Himalaya College of Engineering. IR sensor arrays for line tracking, microcontroller for real-time path decision logic, and H-bridge motor drivers for differential steering. Received Certificate of Appreciation from Himalaya Robotics Club.

Personal Project · IoT Security

ESP32 RFID Access Control & Security Analysis

Personal project extending home automation RFID work into a dedicated access control security study. ESP32-based RFID authentication system with real-time web dashboard — built to identify and document vulnerabilities: replay attacks, lack of mutual authentication, and cleartext credential exposure in low-cost deployments.
